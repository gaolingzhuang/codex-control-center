# Code of Conduct

We want this project to be a practical, welcoming place to collaborate.

Be respectful, focus criticism on the work, assume good intent while asking for evidence, and avoid harassment, discrimination, threats, or sharing another person's private information. Maintainers may edit or remove contributions that violate these expectations and may restrict participation when necessary to protect the community.

If conduct cannot be discussed safely in public, report it privately through the repository's GitHub Security Advisory flow and mark the report as a conduct concern.
